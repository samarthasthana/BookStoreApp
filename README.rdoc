== README
This is a Bookstore App under construction for personal purpose. 

General info:

* Ruby version : 2.1.1

* Rails version : 4.2.2

* Database creation : rake db:create:all

* How to run the test suite: rspec spec/*

* Installation instructions:
  - su bundle install
  - rake db:create:all
  - rake db:migrate
  - rails server
